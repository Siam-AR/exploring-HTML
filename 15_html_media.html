HTML Media Elements

1. Video Element

Attributes used: controls, muted, autoplay, loop, poster

2. Audio Element

Attributes used: controls

3. Image Element

Attributes used: src, alt, width/height

Notes:

  • <video>, <audio>, <img> allow embedding media in HTML.
  • Use multiple <source> for browser compatibility.
  • Always provide fallback text or alt attributes.
